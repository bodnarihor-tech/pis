# Лабораторна робота: Класи та конструктори в C++

## Предметна область
Керування книгами та читачами бібліотеки.

## Що реалізовано
* **Клас Book:** конструктори з параметрами за замовчуванням.
* **Клас Reader:** делегування конструкторів.
* **Клас Librarian:** списки ініціалізації та деструктори.
